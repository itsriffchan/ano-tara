# 🍜 Ano Tara? – Smart Food Picker

A fun, fast, and practical web app inspired by the Filipino **“ano tara”** trend—helping users decide what to eat based on budget, randomness, and nearby availability.

---

## 🚀 Overview

**Ano Tara?** removes the hassle of choosing what to eat by combining:

* Budget-based filtering
* AI-powered food suggestions
* A spinner for random selection
* Seamless integration with Google Maps

Instead of overthinking, just click, spin, and go.

---

## 🎯 Features

### 💸 Budget-Based Options

Users can choose from three price ranges:

* **₱50–₱100**
* **₱100–₱300**
* **₱300+**

This ensures suggestions stay realistic and accessible.

---

### 🤖 AI-Powered Suggestions

The app uses **Gemini AI** to:

* Search for food options within the selected budget
* Generate relevant and trending choices
* Keep suggestions dynamic and varied

---

### 🎡 Spinner Selection

* A visual spinner randomly picks a food item
* Adds a fun, game-like decision process
* Eliminates indecision instantly

---

### 📍 “Ano Tara” Button (Google Maps Integration)

Once a food is selected:

* Clicking **“Ano Tara”** opens Google Maps
* Automatically searches for **nearby locations** offering that food
* Helps users act on the decision immediately

---

## 🧠 How It Works

1. User selects a **budget range**
2. Gemini AI generates **food options**
3. Options are placed into a **spinner**
4. Spinner randomly selects a food
5. User clicks **“Ano Tara”**
6. Redirects to **Google Maps** showing nearby stores/restaurants

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **AI Integration:** Gemini API
* **Maps Integration:** Google Maps (search query redirect)
